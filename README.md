## Cinematic Dialogue / Immersive Interactions
# Overview
This add-on reimagines NPC dialogue interactions in The Elder Scrolls Online to give players greater agency and a more cinematic storytelling experience. It replaces rigid UI-driven conversations with seamless transitions between gameplay and dialogue, inspired by modern single-player RPGs such as The Witcher 3 and Red Dead Redemption 2. This add-on is in active development, with new features added regularly.

## Features
Disables the default interaction camera
Removes UI movement restrictions during interactions
Optional redesign of dialogue presentation for a more cinematic feel
Offers player customizations for dialogue layout and behavior
Adaptive Emotes for roleplaying
Camera Controls for directing scenes
